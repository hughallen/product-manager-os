# Data & Analytics

## Purpose
This folder contains analytics, metrics, data exports, and analysis for this initiative.

## What Goes Here
- **CSV exports** - Data from analytics tools, databases, surveys
- **Analysis documents** - Insights from data analysis
- **Metrics tracking** - KPI dashboards and metric definitions
- **Experiment results** - A/B test results and learnings
- **User feedback** - Aggregated feedback from support, surveys, reviews

## Quick Analysis
Use `@pm-toolkit/8-analytics/quick-insights.mdc` for rapid 30-second analysis:
1. Upload CSV or data file to this folder
2. Run `@pm-toolkit/8-analytics/quick-insights.mdc`
3. Get immediate insights with ASCII visualizations
4. Save analysis document here

## Naming Convention
- `analysis-[topic]-[date].md` - Analysis documents
- `metrics-[date].md` - Metric snapshots
- `experiment-[name]-results.md` - Experiment outcomes
- `[source]-export-[date].csv` - Raw data exports

## Usage
- Reference data in PRDs and strategy docs
- Use for evidence-based decision making
- Track metrics over time
- Validate assumptions with real data
