# Product Specifications

## Purpose
This folder contains product requirements documents (PRDs), one-pagers, and other product specifications.

## What Goes Here
- **PRDs** - Comprehensive product requirements created with `@pm-toolkit/3-planning/create-prd.mdc`
- **One-Pagers** - Concise product summaries created with `@pm-toolkit/3-planning/create-1-pager.mdc`
- **Feature Specs** - Detailed specifications for specific features
- **Technical Requirements** - Any technical specification documents

## Naming Convention
- `prd-[feature-name].md` - Main product requirements
- `one-pager-[feature-name].md` - Executive summaries
- `spec-[feature-name].md` - Feature specifications

## Usage
When creating new specs, use the appropriate guide:
- `@pm-toolkit/3-planning/create-prd.mdc` for full product requirements
- `@pm-toolkit/3-planning/create-1-pager.mdc` for executive summaries

Link to related research, design, and data in other folders.
