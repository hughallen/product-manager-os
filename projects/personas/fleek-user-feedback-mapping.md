## How to Use This for Case Study

**During Data Analysis (24 hours before):**
- Segment users by order size, frequency, purchase history
- Map drop-off points to persona hypotheses
- Validate which persona(s) have activation problem

**During Live Session:**
- Reference specific quotes when presenting problem
- Show which persona(s) each solution addresses
- Acknowledge what we don't know and how you'd validate

**Key Principle:**
Don't assume personas until data confirms. Design solutions that could address multiple personas where possible.
